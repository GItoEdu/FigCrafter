# FigCrafter

FigCrafterは、図や画像を直感的に編集・作成するためのアプリケーションです。

## 主な機能

- **画像のインポートと編集**: ドラッグ＆ドロップで画像を簡単に取り込み、リサイズ、回転、トリミングが可能です。
- **自動ズーム**: 画像をインポートした際、表示領域に合わせて自動的にズーム調整が行われます。
- **ベクトルグラフィックのサポート**: PDFやSVGなどのベクターファイルをインポートし、鮮明な描画を維持したまま編集できます。
- **消しゴムツール**: 非破壊的な消しゴム機能により、画像の一部を透過させることができます。
- **Undo/Redo**: すべての操作を履歴として保持し、自由に行き来できます。

## 使い方

1. `dotnet run` でアプリケーションを起動します。
2. 画像ファイルをウィンドウにドラッグ＆ドロップしてインポートします。
3. 左側のツールバーから描画ツールを選択し、キャンバスを編集します。
4. `Ctrl + 0` でいつでもウィンドウに合わせたズームが可能です。
