# FigCrafter

FigCrafter は、学術論文用の図表（Figure）を多重レイヤーで構成・編集するためのデスクトップアプリケーションです。

## 主な機能
- **多重レイヤー編集**: 複数の画像や図形をレイヤー構造で管理。
- **EMFインポート**: Prism等のグラフ作成ツールから高解像度のベクターデータをインポート。
- **消しゴムツール**: 画像の一部を非破壊で透過させる消しゴム機能。
- **豊富なエクスポートオプション**: PNG, PDF, TIFF等、投稿規定に合わせた出力。

## インストール
本プロジェクトは .NET 8 (WPF) で構築されています。
1. [Inkscape](https://inkscape.org/) をインストールしてください（EMFインポートに必要）。
2. `FigCrafterApp.sln` を Visual Studio 2022 で開き、ビルド・実行してください。

## 開発状況
- **エクスポート解像度**: エクスポート時のスケーリング不具合を修正し、デフォルト **300 DPI** での出力をサポートしました。(`docs/export_scaling_fix/`)
- **ドラッグ&ドロップ / ベクターインポート**: 複数ファイルの一括読み込み、および PDF / EMF のベクター展開をサポート。(`docs/pdf_vector_import/`)
- **データ永続化**: 消しゴム操作のマスクデータ保存に対応しました。
- **画像調整**: 輝度ヒストグラムとスライダーが完全に整列した一元化ダイアログでの調整をサポート。
    - **Undo 最適化**: 調整中のリアルタイムなスタック蓄積を抑え、「適用」時に一括で記録するスマートな Undo 機能を実装。
    - **ショートカット**: `Ctrl+Shift+C` によるクイック起動に対応。
- **UI 刷新とインポート根本修正**: アイコンを `Segoe Fluent Icons` に統一し、PDF インポートの精度（実寸・座標・色、および選択枠の不一致）を大幅に向上させました。 (`docs/image_import_and_ui_icons/`)
