# FigCrafter

FigCrafter は、学術論文用の図表（Figure）を多重レイヤーで構成・編集するためのデスクトップアプリケーションです。

## 主な機能
- **多重レイヤー編集**: 複数の画像や図形をレイヤー構造で管理。
- **EMFインポート**: Prism等のグラフ作成ツールから高解像度のベクターデータをインポート。
- **消しゴムツール**: 画像の一部を非破壊で透過させる消しゴム機能。
- **豊富なエクスポートオプション**: PNG, PDF, TIFF等、投稿規定に合わせた出力。

## インストール
本プロジェクトは .NET 8 (WPF) で構築されています。
1. [Inkscape](https://inkscape.org/) をインストールしてください（EMFインポートに必要）。
2. `FigCrafterApp.sln` を Visual Studio 2022 で開き、ビルド・実行してください。
