# コード変更の解説

## 変更されたファイルと内容

### 1. `Models/GraphicObject.cs`
- **内容**: 新規に `LineObject` クラスを追加しました。
- **目的**: 直線は始点（`X`, `Y`）から終点に向かって描画されるため、終点座標を保持する `EndX`, `EndY` プロパティを定義しました。
- **詳細**: `Draw` メソッドをオーバーライドし、SkiaSharp の `DrawLine` を使って直線を描画する処理（アンチエイリアス有効、線の色指定）を実装しました。

### 2. `Views/CanvasView.xaml.cs`
- **MouseLeftButtonDown の変更**:
  - `DrawingTool.Line` が選択されている際の処理を追加しました。
  - クリックされた座標を始点・終点とする `LineObject` の一時オブジェクトを生成し、操作中の半透明な色（Alpha=128）を設定しました。
- **MouseMove の変更**:
  - ドラッグ中のマウス座標をオブジェクトの幅・高さとして更新する汎用処理では直線が正しく表示できない（マイナス方向などを表現できない）ため、`_tempObject` が `LineObject` かどうかで条件分岐を追加しました。
  - `LineObject` の場合は、マウスの現在位置を直接 `EndX`, `EndY` に代入し、直線の向きと長さをリアルタイムで更新できるようにしました。
- **MouseLeftButtonUp の変更**:
  - マウスのドラッグを終えて直線を確定させる際に、線の色を一時的な半透明色から本番の色（黒色）に変更してキャンバスへオブジェクトを追加するようにしました。
