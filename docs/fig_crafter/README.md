# FigCrafter

## プロジェクト概要
Adobe PhotoshopおよびIllustratorの主要機能を抽出・統合した、軽量で効率的なWindowsデスクトップアプリケーション。

## 技術スタック（検討中）
- **対象OS**: Windows
- **言語**: C# (WPF) または Python (PySide6等)
- **主要機能**:
    - ベクター描画（直線、図形、テキスト、重ね順、グループ化、整列）
    - ラスター処理（レイヤー、透過度、クリッピング、トリミング、回転）
    - ファイル入出力（AI, EMF, PNG, PDF, TIF）
