# UIレイアウト改良と画像DPI/座標系の修正

本タスクでは、FigCrafterのUIレイアウトの抜本的な見直しと、インポートした画像の表示サイズが正しくない（DPIおよび単位系の不一致）問題の解決を行いました。

## 主な変更内容

1. **UIレイアウトの再構成**:
   - メニューバー、プロパティツールバー、図形ツールバーの新設。
   - 右パネルをレイヤー、アートボード設定、色設定に特化。
   - ツールバーボタンのサイズ拡大による操作性向上。

2. **画像インポートの修正**:
   - TIFファイルのインポート対応（System.Drawing経由でのデコード）。
   - インポート時のDPI情報の取得と、物理サイズ（mm）への正確な変換。

3. **キャンバス座標系のミリメートル（mm）単位統一**:
   - キャンバスの描画スケールにミリメートル変換を導入。
   - マウス座標変換をミリメートル基準に統一。
   - 全てのオブジェクト情報をmm単位で一貫して扱えるように修正。
